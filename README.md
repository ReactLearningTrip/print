API功能：
```
1、APP唤起

2、入参支持：http的url链接（pdf、图片）

3、支持图片去除周围白边功能（功能可选）

4、可支持多台打印机

5、可选纸张大小如A4、A5，也可自定义纸张大小

6、可设置打印区域，通过设置边距间接设置

7、自动按图片比例在打印区域水平居中、竖直居中
```
